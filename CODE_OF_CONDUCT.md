# Code of Conduct

This project uses a simple contributor standard:

- Be respectful and constructive.
- Focus criticism on code, behavior, documentation, and project outcomes.
- Do not harass, threaten, or disclose private information.
- Assume public issues and pull requests are visible to future users.

Maintainers may edit, hide, or remove comments and contributions that make the
project unsafe or unproductive for contributors and users.
